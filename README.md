# FDEJECT.X

FD eject utility for X680x0/Human68k

ソフトウェア的にフロッピーをイジェクトします。

なお、キーボード操作可能な状況であれば、CTRL+F1, CTRL+F2 で同じことができます。

---

### インストール方法

EJECT010.ZIP をダウンロードして展開し、FDEJECT.X をパスの通ったディレクトリに置きます。

---

### 使用方法

ドライブ0のイジェクト:

        fdeject 0

ドライブ1のイジェクト:

        fdeject 1
